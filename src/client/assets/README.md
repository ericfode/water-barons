# Waterbarons Asset Pack

The current runtime art pack is modular PNG, loaded by Phaser as image textures.
The SVG files are editable source artifacts; the game imports the PNG files.

Style target:

- bright corporate infrastructure
- ivory shell, teal glass, brass accent, clean shadows
- generated-map compatible tile vocabulary
- no grunge, cyberpunk darkness, fantasy symbols, or board-game token framing

Terrain PNG assets live in `terrain/` and share a `92x48` isometric diamond.
Facility PNG assets live in `facilities/` and share a `128x128` upright silhouette.

Current terrain set:

- `coast`
- `aquifer`
- `fog`
- `ridge`
- `scrub`
- `ruins`
- `sink`
- `hidden`

Current facility set:

- `aquifer-well`
- `desalination-plant`
- `fog-net-array`
- `cloud-harvester`
- `glacial-tap`
- `greywater-loop`
